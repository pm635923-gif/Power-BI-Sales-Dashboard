# 📊 Power BI Sales Dashboard

## 🚀 Project Overview
This project is a Sales Dashboard built using Microsoft Power BI. It analyzes sales performance and includes 30-day forecasting.

## 🔧 Tools Used
- Power BI
- DAX
- Power Query

## 📈 Features
- Sales analysis by region and category
- Interactive filters and slicers
- 30-day sales forecasting
